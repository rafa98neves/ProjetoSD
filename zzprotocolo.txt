<<Protocolo a enviar pelo Servidor RMI:>>

Fazer registo -> type | registo ; username | tintin* ; password | unicorn* 
Fazer login -> type | login ; username | tintin* ; password | unicorn*
Ver se o utilizador tem notificacoes pendentes -> type | notifications ; username | tintin*
Adicionar notificação -> type | add_notification ; username | notificado ; notificacion | "asdasdsa"
encontrar musica/album/artista/genero -> type | search ; name | zeca afonso* ; from | [album/music/genre/artist] 
detalhes musica/album/artista/genero -> type | details; name | zeca afonso* ; from | [album/music/artist]
escrever critica a um album -> type | critic ; username | tintin* ; album | Example* ; write | blablabla*
dar/tirar previlegios -> type | privileges ; username | ze* ; editor | [true/false]
partilhar musica -> type | share ; username | tintin* ; (...)
download_upload musica -> type | GetAddress ; username | tintin*
alterar dados -> type | alteration ; username | tintin* ; [musica/album] | nome ; [artista/genero/...] | alteracao
Adicionar informacao -> type | add ; from | album* ; named | MusicasDoToy* ; this |  CoracaoNaoTemIdade*
Remover informacao -> type | remove ; from | album* ; named | MusicasDoToy* ; this |  RosaNegra*
Get generos -> type | getgeneros
Add Genero -> type | addgenero; genero | rock*

							type | notifications ; notification_1 | none

					type | search ; possibility_1 | none
detalhes musica/album/artista/genero -> type | details ; nome | joao* ; idade | 22 anos*

					type | details ; detail_1 | none
